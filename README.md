# simple2-java

Simulador de la ruta del "ordenador" Simple2 (implementación en Java)
 
El proyecto fin de carrera presentado en octubre de 2003. 

## Construyendo el proyecto

Clonar de github y construir con maven

    git clone https://github.com/albfernandez/simple2-java.git
    cd simple2-java
	mvn clean package

el resultado está en target/simple2-2.0.0.jar, para ejecutarlo
	java -jar target/simple2-2.0.0.jar
